# `Ti.Version`

```js
Version():String
```

Take a glance of current Titanium version.

## `Ti.Version@params`

*No Params*

## `Ti.Version@return`

`String` to indicate current Titanium framework version.

## `Ti.Version@usage`

```js
console.log(Ti.Version())
// "1.0"
```
