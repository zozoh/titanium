# ObjJson Overview

`<ti-obj-json>` can:

- Edit any json object, event `Number|Boolean|String`
- It will support the value type:
  + `Object`
  + `Array`
  + `Number`
  + `Boolean`
  + `String`
  
The GUI base on `ti-tree` with one value field like:
