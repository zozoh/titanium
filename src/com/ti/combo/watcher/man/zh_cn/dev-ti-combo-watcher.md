# 概述

`TiComboWatcher` 可以记录一组搜索和排序条件，然后传递到自己的 `TiFilterbar` 中用来显示

- 搜索条件可以存储到本地

```bash
[*Condition A*] [Condition B] [Condition C]  +  <-- 复制当前条件
+---------------------------------------------------------+
| Filter Bar                                              |
+---------------------------------------------------------+

```
