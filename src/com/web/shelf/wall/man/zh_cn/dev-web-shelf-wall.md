# 墙概述

将传入数据，用数据渲染控件渲染成墙贴。

墙贴布局有下面几种方式

## 水平布局:`horizontal`

## 垂直布局:`vertical`

# 属性列表

| Name       | Description |
| ---------- | ---------------- |
| `data`     | 【数组】控件数据 |
| `comType`  | 墙贴项控件类型 |
| `comConf`  | 墙贴项控件配置 |
| `blankAs`  | 空白数据显示方式 |
| `loadingAs`| 加载数据显示方式 |
